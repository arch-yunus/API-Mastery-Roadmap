# 06. Infrastructure & CI/CD: Orchestrating the Ecosystem

An API is only as good as the infrastructure that hosts and monitors it.

## 📖 Topics Covered

### 1. API Gateways & Service Mesh
*   **Gateways**: Edge routing, Auth offloading, and Load balancing (Kong, Tyk).
*   **Service Mesh**: Inter-service communication and mutual TLS (Istio, Linkerd).

### 2. Observability & Monitoring
*   **Tracing**: Distributed tracing with Jaeger or Zipkin.
*   **Logging**: Centralized aggregation (ELK Stack, Datadog).
*   **Metrics**: Golden signals (Latency, Traffic, Errors, Saturation) with Prometheus.

### 3. CI/CD & Automation
*   **SDK Generation**: Generating client libraries from OpenAPI specs.
*   **Breaking Change Detection**: Continuous integration tests for backward compatibility.
*   **Deployment Strategies**: Blue/Green and Canary deployments for APIs.
